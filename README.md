# irodori-knowledge

Shared error, job, and knowledge-store crates for Irodori.

## Crates

- `irodori-error`: shared error type.
- `irodori-jobs`: progress, cancellation, retries, and checkpoints.
- `irodori-knowledge`: SQLite-backed source snapshots and search.

## Develop

cargo test --workspace
